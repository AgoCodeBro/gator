# Gator

## Description

Gator is a CLI tool that is an RSS aggregator in Go. It will allow users to:

- Add RSS feeds
- Store collected posts in a database
- Follow and unfollow other users feeds
- View summaries of aggregated posts in the CLI along with a link to said posts

## Learning Goals

This project is meant as learning exercise to help practice

- Learn how to integrate Go with a SQL database
- Practice using SQL using sqlc and goose in Go
- Learn how to write a long-running service that continuously fetches new posts from RSS feeds and stores them in the database
